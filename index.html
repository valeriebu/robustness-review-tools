Robustness & Transparency Scoring Sheet
Score each study on 7 robustness dimensions (R1–R7) and 7 transparency dimensions (T1–T7). Yes/no per question, tallied into % scores per dimension. Transfer scores to the radar chart tool. File: Robustness_Transparency_Scoring_Sheet.docx

Results Across Subgroups — Step 2
Records clinical outcome results and AI performance metrics broken down by subgroup (sex, age, race, clinical group, setting, etc.). Flags where subgroup analyses were not reported. File: Robustness_Step2_Results_Across_Subgroups.docx

Tool-Level Overview Sheet
One sheet per tool — aggregates findings across the RCT and all precursor studies. Rows = dimensions, columns = each study + tool-level conclusion. Scores plus space to write what was found per cell. File: Tool_Overview_Sheet.docx
How to use these tools together
1. Map the studies found describing a specific tool. — Open the Review Overview tool. Add each AI tool and its associated studies. This gives you a bird's-eye view of what you're working with.

2. Extract information of each study. — Use the Extraction Codebook (Word) per study. Amber-highlighted questions are robustness-relevant. This might become automatised in the future.

3. Visualise validation evidence — Use the Validation Cascade tool to plot performance across all 5 validation steps per tool. See where performance drops and how deep the evidence goes. Export as PNG.

4. Visualise robustness & transparency profiles — Enter scores into the Radar Chart tool. Export as PNG for your paper.

5. Summarise per tool — Complete the Tool-Level Overview Sheet (Word) to aggregate evidence across the RCT and precursors into a single tool-level judgement. (IN PROGESS!)
All tools run entirely in your browser — no data is sent to any server. To update a tool, replace the corresponding HTML file and refresh the page.
